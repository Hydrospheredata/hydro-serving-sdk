hydrosdk.application module¶

class hydrosdk.application.Application(cluster, id, name, execution_graph, status, signature, kafka_streaming, metadata=None)¶

Bases: object

Applications are used to combine your ModelVersions into a linear graph and deploy it into production, exposing HTTP and gRPC interfaces for consumers.

Use ApplicationBuilder class to create a new Application in your Hydrosphere cluster or an Application.find method to get the existing Application.

Application

List all applications created on the cluster.

>>> from hydrosdk.cluster import Cluster
>>> cluster = Cluster("http-cluster-endpoint")
>>> apps = Application.list(cluster)
>>> for app in apps:
>>>     print(app)

Find an application by name and perform a prediction from it.

>>> from hydrosdk.cluster import Cluster
>>> cluster = Cluster("http-cluster-endpoint", "grpc-cluster-endpoint")  # important to use a gRPC endpoint
>>> app = Application.find(cluster, "my-application")
>>> pred = app.predictor()
>>> resp = pred.predict({"my-input": 1})

  • id (int) – unique application ID

  • name (str) – application Name

  • signature (hydro_serving_grpc.contract.model_signature_pb2.ModelSignature) – signature, specifying input and output fields names, dtypes and shapes

  • execution_graph (ExecutionGraph) – linear graph which specifies ExecutionStages which sequentially transform input

  • status (hydrosdk.application.ApplicationStatus) – Application Status

  • kafka_streaming (List[hydrosdk.application.StreamingParams]) – list of Kafka parameters with input and output Kafka topics specified

  • metadata (Optional[Dict[str, str]]) – metadata with string keys and string values.

class hydrosdk.application.ApplicationBuilder(cluster, name)¶

Bases: object

ApplicationBuilder is used to create new Applications in your cluster.

Example

Parameters

  • cluster (hydrosdk.cluster.Cluster) –

  • name (str) –

Return type

ApplicationBuilder

Create an application from existing modelversions.

>>> from hydrosdk import Cluster, ModelVersion
>>> cluster = Cluster('http-cluster-endpoint')
>>> mv1 = ModelVersion.find(cluster, "my-model", 1)
>>> mv2 = ModelVersion.find(cluster, "my-model", 2)
>>> stage = ExecutionStageBuilder()                     .with_model_variant(mv1, 50)                     .with_model_variant(mv2, 50) \
                .build()
>>> app = ApplicationBuilder(cluster, "my-application-ab-test")                 .with_stage(stage)                 .build()

  • cluster (hydrosdk.cluster.Cluster) – Hydrosphere cluster where you want to create an Application

  • name (str) – Future Application name
